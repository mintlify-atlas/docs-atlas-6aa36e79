## Overview

The TallerXEmpresa API manages the many-to-many relationship between branding workshops and companies. This allows companies to have preferred workshops for their campaign vehicle branding needs, and workshops to serve multiple company clients.

## The TallerXEmpresa Object

<ResponseField name="id_taller" type="integer" required>
  Unique identifier for the workshop-company relationship
</ResponseField>

<ResponseField name="id_empresa" type="integer" required>
  Foreign key reference to the company (Empresa)
</ResponseField>

<ResponseField name="estado" type="integer" required>
  Relationship status:
  - `0` - Inactive/Terminated
  - `1` - Active
  - `2` - Suspended
  - `3` - Pending approval
</ResponseField>

<ResponseField name="fecha_creacion" type="date" required>
  Date when the relationship was established (YYYY-MM-DD format)
</ResponseField>

<ResponseField name="fecha_modificacion" type="date" required>
  Date when the relationship was last modified (YYYY-MM-DD format)
</ResponseField>

## Create Workshop-Company Relationship

<RequestExample>
```bash cURL
curl -X POST https://api.restapi.com/taller-empresa \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "id_empresa": 5,
    "estado": 1,
    "fecha_creacion": "2026-03-09",
    "fecha_modificacion": "2026-03-09"
  }'
```

```python Python
import requests

url = "https://api.restapi.com/taller-empresa"
headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
}
data = {
    "id_empresa": 5,
    "estado": 1,
    "fecha_creacion": "2026-03-09",
    "fecha_modificacion": "2026-03-09"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
```

```javascript JavaScript
fetch('https://api.restapi.com/taller-empresa', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    id_empresa: 5,
    estado: 1,
    fecha_creacion: '2026-03-09',
    fecha_modificacion: '2026-03-09'
  })
})
.then(response => response.json())
.then(data => console.log(data));
```
</RequestExample>

<ResponseExample>
```json Response
{
  "id_taller": 1,
  "id_empresa": 5,
  "estado": 1,
  "fecha_creacion": "2026-03-09",
  "fecha_modificacion": "2026-03-09"
}
```
</ResponseExample>

<ParamField body="id_empresa" type="integer" required>
  Company ID to associate with the workshop
</ParamField>

<ParamField body="estado" type="integer" required>
  Status of the relationship (0: Inactive, 1: Active, 2: Suspended, 3: Pending)
</ParamField>

<ParamField body="fecha_creacion" type="date" required>
  Date when the relationship was created in YYYY-MM-DD format
</ParamField>

<ParamField body="fecha_modificacion" type="date" required>
  Last modification date in YYYY-MM-DD format
</ParamField>
