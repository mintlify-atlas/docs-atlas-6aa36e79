## Overview

The EntidadBancaria API manages banking entities and financial institutions used for processing payments and managing financial transactions within the platform.

## The EntidadBbancaria Object

<ResponseField name="id_entidad" type="integer" required>
  Unique identifier for the banking entity
</ResponseField>

<ResponseField name="id_ciudad" type="integer" required>
  Foreign key reference to the city where the banking entity operates
</ResponseField>

<ResponseField name="id_pais" type="integer" required>
  Foreign key reference to the country where the banking entity is located
</ResponseField>

<ResponseField name="nombre" type="string" required>
  Name of the banking entity (max 200 characters)
</ResponseField>

## Create Banking Entity

<RequestExample>
```bash cURL
curl -X POST https://api.restapi.com/entidades-bancarias \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "id_ciudad": 1,
    "id_pais": 1,
    "nombre": "Banco Pichincha"
  }'
```

```python Python
import requests

url = "https://api.restapi.com/entidades-bancarias"
headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
}
data = {
    "id_ciudad": 1,
    "id_pais": 1,
    "nombre": "Banco Pichincha"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
```

```javascript JavaScript
fetch('https://api.restapi.com/entidades-bancarias', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    id_ciudad: 1,
    id_pais: 1,
    nombre: 'Banco Pichincha'
  })
})
.then(response => response.json())
.then(data => console.log(data));
```
</RequestExample>

<ResponseExample>
```json Response
{
  "id_entidad": 1,
  "id_ciudad": 1,
  "id_pais": 1,
  "nombre": "Banco Pichincha"
}
```
</ResponseExample>

<ParamField body="id_ciudad" type="integer" required>
  City ID where the banking entity operates
</ParamField>

<ParamField body="id_pais" type="integer" required>
  Country ID where the banking entity is registered
</ParamField>

<ParamField body="nombre" type="string" required>
  Full name of the banking entity (max 200 characters)
</ParamField>

## Update Banking Entity

<RequestExample>
```bash cURL

  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "nombre": "Banco Pichincha S.A.",
    "id_ciudad": 2
  }'
```

```python Python
import requests

url = "https://api.restapi.com/entidades-bancarias/1"
headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
}
data = {
    "nombre": "Banco Pichincha S.A.",
    "id_ciudad": 2
}

response = requests.put(url, json=data, headers=headers)
print(response.json())
```

