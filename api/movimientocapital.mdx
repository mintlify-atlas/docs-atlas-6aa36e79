## Overview

The MovimientoCapital API manages financial transactions for advertising campaigns, tracking deposits, withdrawals, and accumulated balances for clients participating in campaigns.

## The MovimientoCapital Object

<ResponseField name="id_saldo" type="integer" required>
  Unique identifier for the capital movement transaction
</ResponseField>

<ResponseField name="id_campana" type="integer" required>
  Foreign key reference to the advertising campaign (CampanaPublicitaria)
</ResponseField>

<ResponseField name="tipo_transaccion" type="integer" required>
  Transaction type identifier:
  - `1` - Deposit/Credit
  - `2` - Withdrawal/Debit
  - `3` - Refund
  - `4` - Adjustment
</ResponseField>

<ResponseField name="descripcion" type="string" required>
  Brief description of the transaction (max 10 characters)
</ResponseField>

<ResponseField name="id_cliente" type="integer" required>
  Foreign key reference to the client making the transaction
</ResponseField>

<ResponseField name="fecha_transaccion" type="date" required>
  Date when the transaction occurred (YYYY-MM-DD format)
</ResponseField>

<ResponseField name="monto_transaccion" type="float" required>
  Transaction amount in USD
</ResponseField>

<ResponseField name="saldo_acumulado" type="float" required>
  Cumulative balance after this transaction
</ResponseField>

<ResponseField name="id_ciudad" type="integer" required>
  Foreign key reference to the city where the transaction was processed
</ResponseField>

<ResponseField name="id_pais" type="integer" required>
  Foreign key reference to the country where the transaction was processed
</ResponseField>

<ResponseField name="estado" type="integer" required>
  Transaction status:
  - `0` - Cancelled
  - `1` - Active/Completed
  - `2` - Pending
  - `3` - Failed
</ResponseField>

## Create Capital Movement

<RequestExample>
```bash cURL
curl -X POST https://api.restapi.com/movimiento-capital \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "id_campana": 15,
    "tipo_transaccion": 1,
    "descripcion": "Deposit",
    "id_cliente": 42,
    "fecha_transaccion": "2026-03-09",
    "monto_transaccion": 1500.00,
    "saldo_acumulado": 1500.00,
    "id_ciudad": 1,
    "id_pais": 1,
    "estado": 1
  }'
```

```python Python
import requests

url = "https://api.restapi.com/movimiento-capital"
headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
}
data = {
    "id_campana": 15,
    "tipo_transaccion": 1,
    "descripcion": "Deposit",
    "id_cliente": 42,
    "fecha_transaccion": "2026-03-09",
    "monto_transaccion": 1500.00,
    "saldo_acumulado": 1500.00,
    "id_ciudad": 1,
    "id_pais": 1,
    "estado": 1
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
```

```javascript JavaScript
fetch('https://api.restapi.com/movimiento-capital', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',

```

<ParamField body="id_campana" type="integer" required>
  Campaign ID for which the transaction is being recorded
</ParamField>

<ParamField body="tipo_transaccion" type="integer" required>
  Type of transaction (1: Deposit, 2: Withdrawal, 3: Refund, 4: Adjustment)
</ParamField>

<ParamField body="descripcion" type="string" required>
  Short description of the transaction (max 10 characters)
</ParamField>

<ParamField body="id_cliente" type="integer" required>
  Client ID associated with this transaction
</ParamField>

<ParamField body="fecha_transaccion" type="date" required>
  Transaction date in YYYY-MM-DD format
</ParamField>

<ParamField body="monto_transaccion" type="float" required>
  Transaction amount (positive for deposits, negative for withdrawals)
</ParamField>

<ParamField body="saldo_acumulado" type="float" required>
  Running balance after applying this transaction
</ParamField>

<ParamField body="id_ciudad" type="integer" required>
  City ID where the transaction occurred
</ParamField>

<ParamField body="id_pais" type="integer" required>
  Country ID where the transaction occurred
</ParamField>

<ParamField body="estado" type="integer" required>
  Transaction status (0: Cancelled, 1: Active, 2: Pending, 3: Failed)
</ParamField>
