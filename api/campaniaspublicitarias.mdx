## Overview

The Campañas Publicitarias (Advertising Campaigns) endpoint allows you to create and manage advertising campaigns that target specific vehicles within defined geographic sectors. Campaigns include budget management, vehicle type eligibility, branding specifications, and monetization schedules.

## Base URL

```
/api/campaniaspublicitarias/
```

## Endpoints

### List All Campaigns

<api-endpoint endpoint="GET /api/campaniaspublicitarias/" />

Retrieve a list of all advertising campaigns.

**Response**

<ResponseField name="id_campana" type="integer">
  Unique identifier for the campaign (auto-generated)
</ResponseField>

<ResponseField name="id_empresa" type="integer">
  Foreign key to the company (Empresa) that owns this campaign
</ResponseField>

<ResponseField name="id_ciudad" type="integer">
  Foreign key to the city where the campaign operates
</ResponseField>

<ResponseField name="id_pais" type="integer">
  Foreign key to the country where the campaign operates
</ResponseField>

<ResponseField name="nombre_campana" type="string">
  Name of the campaign (max 20 characters)
</ResponseField>

<ResponseField name="correo_responsable" type="string">
  Email address of the campaign manager (max 40 characters)
</ResponseField>

<ResponseField name="id_sector" type="integer">
  Reference to the sector ID (nullable)
</ResponseField>

<ResponseField name="fecha_inicio" type="date">
  Campaign start date (format: YYYY-MM-DD)
</ResponseField>

<ResponseField name="fecha_fin" type="date">
  Campaign end date (format: YYYY-MM-DD)
</ResponseField>

<ResponseField name="fecha_fin_registro" type="date">
  Registration deadline for drivers to join the campaign (format: YYYY-MM-DD)
</ResponseField>

<ResponseField name="presupuesto" type="float">
  Total budget allocated for the campaign
</ResponseField>

<ResponseField name="nombre_responsable" type="string">
  Name of the person responsible for the campaign (max 20 characters)
</ResponseField>

<ResponseField name="tarifa_base" type="float">
  Base rate for driver compensation
</ResponseField>

<ResponseField name="tarifa_min" type="float">
  Minimum compensation rate
</ResponseField>

<ResponseField name="tarifa_max" type="float">
  Maximum compensation rate
</ResponseField>

<ResponseField name="hora_monetizable_inicio" type="string">
  Start time for monetizable hours (format: HH:MM, max 5 characters)
</ResponseField>

<ResponseField name="hora_monetizable_fin" type="string">
  End time for monetizable hours (format: HH:MM, max 5 characters)
</ResponseField>

<ResponseField name="cobro_minimo" type="float">
  Minimum charge amount
</ResponseField>

<ResponseField name="tipo_brandeo" type="string">
  Type of vehicle branding. Choices: `sticker` or `panel led`
</ResponseField>

<ResponseField name="id_talleres" type="json">
  JSON array of workshop IDs where branding can be applied
</ResponseField>

<ResponseField name="carroceria_capo" type="float">
  Rate for hood branding (default: 0.0)
</ResponseField>

<ResponseField name="carroceria_techo" type="float">
  Rate for roof branding (default: 0.0)
</ResponseField>

<ResponseField name="puerta_conductor" type="float">
  Rate for driver door branding (default: 0.0)
</ResponseField>

<ResponseField name="puerta_pasajero" type="float">
  Rate for passenger door branding (default: 0.0)
</ResponseField>

<ResponseField name="puerta_trasera_iz" type="float">
  Rate for left rear door branding (default: 0.0)
</ResponseField>

<ResponseField name="puerta_trasera_der" type="float">
  Rate for right rear door branding (default: 0.0)
</ResponseField>

<ResponseField name="puerta_maletero" type="float">
  Rate for trunk/boot branding (default: 0.0)
</ResponseField>

<ResponseField name="fecha_creacion" type="date">
  Date when the campaign was created (format: YYYY-MM-DD)
</ResponseField>

<ResponseField name="fecha_modificacion" type="date">
  Date when the campaign was last modified (format: YYYY-MM-DD)
</ResponseField>

<ResponseField name="estado" type="integer">
  Campaign status (0: inactive, 1: active)
</ResponseField>

<ResponseField name="sedan_admisible" type="boolean">
  Whether sedan vehicles are eligible for this campaign
</ResponseField>

<ResponseField name="suv_admisible" type="boolean">
  Whether SUV vehicles are eligible for this campaign
</ResponseField>

<ResponseField name="camion_admisible" type="boolean">
  Whether truck vehicles are eligible for this campaign
</ResponseField>

<ResponseField name="camioneta_admisible" type="boolean">
  Whether pickup truck vehicles are eligible for this campaign
</ResponseField>

<ResponseField name="bus_admisible" type="boolean">
  Whether bus vehicles are eligible for this campaign
</ResponseField>

### Retrieve a Campaign

<api-endpoint endpoint="GET /api/campaniaspublicitarias/{id_campana}/" />

Retrieve details of a specific campaign by ID.

### Update a Campaign

<api-endpoint endpoint="PUT /api/campaniaspublicitarias/{id_campana}/" />

Update all fields of a specific campaign.

<api-endpoint endpoint="PATCH /api/campaniaspublicitarias/{id_campana}/" />

Partially update specific fields of a campaign.

### Delete a Campaign

<api-endpoint endpoint="DELETE /api/campaniaspublicitarias/{id_campana}/" />

Delete a specific campaign by ID.
